                                🎙️
                                
Speech Generation

                                Core API
                            

                            

                                
                                

                                    

                                        POST
                                        /speak
                                    

                                    

                                        
Generate speech using a registered speaker preset. Supports emotion control and text splitting.

                                        

                                            

                                                
Required Parameters

                                                
text string — Text to synthesize

                                                
name string — Speaker preset name

                                            

                                            

                                                
Optional Parameters

                                                
emotion_text string — Emotion description (e.g., "happy and excited")

                                                
emotion_weight float — Emotion intensity 0.0-1.0 (default: 0.6)

                                                
diffusion_steps int — Quality steps (default: 10)

                                                
max_text_tokens_per_sentence int — Text splitting threshold 80-200 (default: 120)

                                            

                                        

                                        

                                            
Response

                                            
Content-Type: audio/mpeg
Binary MP3 audio data

                                        

                                    

                                


                                
                                

                                    

                                        POST
                                        /clone_voice
                                    

                                    

                                        
Clone voice using uploaded reference audio (zero-shot voice cloning).

                                        

                                            

                                                
Required Parameters (multipart/form-data)

                                                
text string — Text to synthesize

                                                
reference_audio_file file — Reference audio for voice cloning

                                            

                                            

                                                
Optional Parameters

                                                
emotion_text, emotion_weight, diffusion_steps, max_text_tokens_per_sentence — Same as /speak

                                            

                                        

                                    

                                


                                
                                

                                    

                                        POST
                                        /speak_stream
                                    

                                    

                                        
Generate speech with streaming chunks for low-latency playback.

                                        

                                            

                                                
Parameters

                                                
Same as /speak

                                            

                                        

                                        

                                            
Streaming Response Format

                                            
CHUNK:{idx}:{size}:{status}\n{audio_bytes}

Status: CONTINUE (more chunks) or LAST (final chunk)

                                        

                                    

                                


                                
                                

                                    

                                        POST
                                        /clone_voice_stream
                                    

                                    

                                        
Clone voice with streaming output. Same parameters as /clone_voice, same streaming format as /speak_stream.

Speech Translation

                                Gemini-Powered
                            

                            

                                
                                

                                    

                                        POST
                                        /api/translate_audio
                                    

                                    

                                        
One-shot speech translation. Translates audio and regenerates voice in the target language while preserving timing.

                                        

                                            

                                                
Required (multipart/form-data or JSON)

                                                
audio_file file — Source audio file

                                                
dest_language string — Target language (e.g., "English", "Chinese")

                                            

                                                

                                                
Audio Separation (Vocal/Instrumental)

                                                
audio_separator_enabled bool — Enable audio-separator for vocal/instrumental separation (recommended for backing track)

                                                
audio_separator_model string — Model: 'fast' (UVR-MDX-NET), 'balance' (default, BS-Roformer), or 'quality' (Mel-Band-Roformer)

                                            

                                                

                                                
ClearVoice Enhancement

                                                
enhance_voice bool — Enable voice enhancement (applied to vocals)

                                                
enhancement_model string — Model: 'MossFormerGAN_SE_16K' (default), 'FRCRN_SE_16K', or 'MossFormer2_SE_48K'

                                                
super_resolution_voice bool — Enable 48kHz super resolution

                                                
merge_backing_track bool — Blend with instrumental (requires audio_separator_enabled)

                                            

                                            

                                                
Advanced Options

                                                
prompt string — Custom Gemini instructions

                                                
segments_json string — Pre-generated Gemini segments (skip inference)

                                                
min_speech_ms / max_merge_ms int — Segment merging heuristics

                                            

                                        

                                        

                                            
Response Headers

                                            
X-Translation-Model: gemini-3.1-flash-lite-preview
X-Translation-Segments: [{"start": "00:00", "end": "00:05", ...}]
Content-Type: audio/mpeg

                                        

                                    

                                


                                
                                

                                    

                                        POST
                                        /api/translate_segments
                                    

                                    

                                        
Prepare editable translation segments for the advanced translate/edit workflow. Returns a session with diarized, editable segments.

                                        

                                            

                                                
Key Parameters

                                                
audio_file file — Source audio

                                                
dest_language string — Target language (required)

                                                
translate_text bool — true=translate, false=transcribe only

                                                
gemini_model string — Model selection (flash/pro)

                                                
reuse_session_id string — Reuse audio from previous session

                                                
default_speaker_preset string — Default voice for all speakers

                                            

                                        

                                        

                                            
SSE Stream Events

                                            
{"event": "status", "stage": "gemini", "message": "..."}
{"event": "complete", "session_id": "...", "segments": [...], "metadata": {...}}

                                        

                                    

                                


                                
                                

                                    

                                        POST
                                        /api/translate_generate_segments
                                    

                                    

                                        
Generate translated audio from edited segments. Supports per-segment speaker assignment and selective regeneration.

                                        

                                            

                                                
JSON Body

                                                
session_id string — Session from translate_segments

                                                
segments array — Edited segment objects

                                                
speaker_overrides object — Map speaker IDs to presets

                                            

                                        

                                        

                                            
Final Event

                                            
{"event": "complete", "audio_url": "/api/translate_outputs/...",
 "subtitle_url": "...", "original_subtitle_url": "..."}

                                        

                                    

                                


                                
                                

                                    

                                        POST
                                        /api/translate_segment_preview
                                    

                                    

                                        
Generate a quick inline preview for a single translated segment before full generation.

                                        

                                            

                                                
JSON Body

                                                
session_id string

                                                
segment object — {index, type, start_ms, end_ms, translated_text, speaker, ...}

                                            

                                        

                                    

                                

                            

                        


                        
                        

                            

                                📦
                                
Batch Audio Processing

                                Long Audio
                            

                            

                                
                                

                                    

                                        POST
                                        /api/translate_split_audio
                                    

                                    

                                        
Split long audio into ClearVoice-enhanced chunks at silence points for parallel processing.

                                        

                                            

                                                
Chunk Settings

                                                
chunk_min_minutes float — Minimum chunk length (default: 3)

                                                
chunk_max_minutes float — Maximum chunk length (default: 6)

                                                
min_silence_ms int — Minimum silence for split (default: 2000)

                                                
silence_threshold_db float — Silence detection threshold (default: -40)

                                            

                                        

                                        

                                            
Final Event

                                            
{"event": "complete", "chunk_batch_id": "...",
 "chunk_sessions": [{"session_id": "...", "chunk_index": 0, ...}]}

                                        

                                    

                                


                                
                                

                                    

                                        POST
                                        /api/translate_generate_chunks
                                    

                                    

                                        
Generate translated audio for multiple chunk sessions in parallel.

                                        

                                            

                                                
JSON Body

                                                
chunk_session_ids array — Array of session IDs to process

                                                
dest_language string — Target language

                                            

                                        

                                    

                                


                                
                                

                                    

                                        POST
                                        /api/translate_merge_chunks
                                    

                                    

                                        
Merge processed chunk sessions back into a single audio file with SRT subtitles.

                                        

                                            

                                                
JSON Body (one of)

                                                
chunk_session_ids array — Ordered list of session IDs

                                                
chunk_batch_id string — Auto-resolve all chunks from batch

                                                
merge_backing_track bool — Include instrumental track

                                            

                                        

                                        

                                            
Response

                                            
{"status": "ok", "audio_url": "...", "subtitle_url": "...",
 "original_subtitle_url": "...", "file_name": "output_english.mp3"}

Basic Speech Generation

                            
# Python
import requests

response = requests.post("http://localhost:8080/speak", json={
    "text": "Hello, how are you today?",
    "name": "speaker1",
    "emotion_text": "cheerful and friendly",
    "emotion_weight": 0.7
})

with open("output.mp3", "wb") as f:
    f.write(response.content)


                            
One-Shot Translation

                            
# Python
import requests

with open("source_audio.mp3", "rb") as f:
    response = requests.post(
        "http://localhost:8080/api/translate_audio",
        files={"audio_file": f},
        data={
            "dest_language": "English",
            "enhance_voice": "true",
            "merge_backing_track": "true"
        }
    )

with open("translated.mp3", "wb") as f:
    f.write(response.content)
